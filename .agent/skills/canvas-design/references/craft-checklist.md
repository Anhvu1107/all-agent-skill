# Craft Checklist

Use this before exporting the canvas artifact.

## Required Checks

- no text or graphics fall outside the canvas bounds
- spacing feels deliberate and consistent
- typography looks chosen and integrated with the composition
- the palette is specific enough that it would feel wrong on a different topic
- the piece reads as designed by a human, not merely decorated by an AI

## When Typography Matters

- search `canvas-fonts/` before defaulting to a generic system font
- use font contrast to reinforce the concept, not just to add variety
