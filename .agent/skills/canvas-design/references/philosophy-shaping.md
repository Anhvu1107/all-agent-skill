# Philosophy Shaping

Use this before generating the design manifesto.

## Turn The Prompt Into A Movement

- extract the quiet conceptual thread instead of literalizing the subject
- choose one dominant visual language: brutalist, chromatic, analog, geometric, organic, or another coherent direction
- decide how the reference shows up through form, rhythm, palette, and texture rather than exposition

## Quality Bar

- the philosophy should create room for interpretation, not a paint-by-numbers layout
- repeated emphasis on craftsmanship should influence the execution standard, not become filler prose
- text should stay sparse and behave like a visual element
