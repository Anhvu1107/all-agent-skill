# Research Routing

Use this before running the broader last-30-days workflow.

## Route By Intent

- prompting question: focus on concrete techniques, repeated patterns, and copy-paste examples
- recommendation question: focus on named tools, products, and repeated community mentions
- news question: focus on what changed, dates, and cross-source corroboration
- general topic scan: build a concise map of themes, debates, and notable examples

## Source Mix

- Reddit for grounded user discussion
- X for fast-moving discourse and launches
- web results for announcements, docs, and broader context
