# Output Shaping

Use this before returning the final research summary.

## Required Shape

- summarize what changed or repeated across sources
- separate raw signal from your synthesis
- include concrete examples, prompts, or named recommendations when the user asked for them
- stay explicit about the recency window

## Avoid

- generic summaries with no source-aware insight
- claiming a trend from one loud example
- mixing old evergreen advice into a "last 30 days" answer without labeling it
