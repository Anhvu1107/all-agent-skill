# Theme Selection

Use this before applying a theme.

## Route By Artifact

- slide deck: prioritize contrast, display typography, and presentation rhythm
- report or document: prioritize long-form readability and restrained accents
- landing page or web artifact: prioritize brand fit, hierarchy, and screen contrast

## Choose A Theme

- match audience and tone before picking colors
- prefer one strong visual direction over a generic compromise
- if no preset really fits, generate a custom theme instead of forcing the wrong preset
