# Workbook Routing

Use this to choose the right spreadsheet workflow.

## Route By Starting Point

- Existing workbook with formulas or formatting: preserve the workbook and edit surgically.
- Messy CSV or TSV cleanup: normalize the data first, then write an Excel-friendly output.
- New workbook or model: decide whether the real deliverable is analysis, reporting, or financial modeling before shaping sheets.

## Route By Risk

- Financial model: keep assumptions explicit and references audit-friendly.
- Operational workbook: prioritize formulas, filters, and readability over decorative formatting.
- Conversion task: preserve headers, data types, and row counts before adding polish.

Always recalculate before claiming the workbook is correct.
