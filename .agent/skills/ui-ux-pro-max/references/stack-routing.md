# Stack Routing

Choose the stack search file that matches the implementation surface:

- `react`, `nextjs`, `remix`, `astro`, `svelte`, `sveltekit`, `vue`, `nuxtjs`, `nuxt-ui`, `shadcn` for web
- `expo-router`, `react-native`, `flutter`, `swiftui`, `jetpack-compose` for native or cross-platform app UI
- `html-tailwind` when the stack is unspecified or the task is plain HTML/CSS utility-first work

If the user names a framework, use that stack first before falling back to generic design advice.
