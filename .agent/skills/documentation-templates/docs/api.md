# API Reference Template

Use this as a starting point when a project needs a lightweight API reference page.

## Endpoint Template

```markdown
## METHOD /path

Short description.

### Parameters

| Name | Type | Required | Description |
|------|------|----------|-------------|
| id | string | Yes | Example parameter |

### Responses

- `200`: Success
- `400`: Validation error
- `404`: Not found
```
