# Routing

Use this before loading language-specific SDK material.

## Route By Surface

- Python project: read `python/`
- TypeScript or Node project: read `typescript/`
- cURL or raw HTTP request: read `curl/`
- other supported languages: read the matching language folder

## Route By Job

- new integration: start with shared defaults and the language folder
- migration: check `shared/` first for model and feature changes
- provider-mismatch suspicion: stop and confirm before editing
